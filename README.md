# prueba_AA_Android

 **************        GARAGE ADMIN        *************

La apliacacion movil que cualquier taller quisiera tener.
Esta aplicacion esta encarada al uso en talleres mecanicos, ya que esta compuesta por diversas funcionalidades que hacen que llevar 
los datos del taller sea mas comodo, sencillo y rapido.

Esta compuesta por una pantalla principal bien diferenciada, en la que podemos tener acceso (mediante contrseñas), o bien al apartado de usuarios,
 o bien al apartado de vehiculos. Separandolo asi conseguidos tener las dos partes bien diferenciadas, ya que puede ser que ciertos usuarios del
 taller deban/puedan tener acceso a ambos departamentos o solo a alguno de ellos.
 
 En la parte de usuarios nos encontramos con las opciones de clientes y de trabajadores. En ambos ellos podemos dar de alta, visualizar, eliminar, editar y 
 buscar a traves de un buscador eligiendo el parametro a traves del cual queremos buscar.
 
 Por el otro lado, en el lado de los vehiculos, podemos realizar las mismas funciones que las anteriormente dichas, pero podemos añadir la funcionalidad 
 de que podemos crear facturas relacionadas directamente con el cliente y el vehiculo implicado.
 
 Como opcion extra, podemos consultar a traves de un mapa los distintos talleres mas importantes del pais, dependiendo de si buscamos repuestos
 de caracter preofesional y para particulares, pudiendo asi ver la distancia a cada uno desde nuestra ubicacion y poder sabe rque pedido de dicho
  taller nos llegaria antes.
